# Simple_Calculator-
A simple Python CLI calculator that performs basic arithmetic operations with clean and structured output.
A Python-based command-line calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.
# Features
- Supports four basic operations
- User-friendly CLI interface
- Handles invalid input and division by zero
- Clean and structured output display
# Purpose
This project is built to practice fundamental Python concepts such as conditionals, user input, and basic problem-solving.
